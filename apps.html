Technical Applications

The infrastructure for Python applications is in sleep mode until you wake it up - please be patient when you click the links.

Analyzing Federal Funding for the Great Lakes Restoration Initiative

This interactive Python application can be used to evaluate the changes in federal funding to the Great Lakes Restoration Initiative based on location, agency, and focus area.

EIA Energy Flow Sankey Diagram

This Python web app models the annual US energy flow Sankey diagram distributed by the Energy Information Administration and demonstrates how it might be automated.

EPIC Water Systems Consolidation Analysis

This Tableau visualization was built for the Environmental Policy Innovation Center with data from the EPA as part of an analysis conducted on the consolidation of water utilities in the United States.

Great Lakes Water Levels, 1918-Present

This reactive web tool was built in R to evaluate changes in the water levels of the Great Lakes. It automatically sources data each month from the NOAA GLERL database.

The Marches of Hannibal and Napoleon: Replicating Charles Minard's Famous Charts in R

This web application was built in R and demonstrates a modern replication of Charles Minard's seminal work in data visualization.

Invasive Species in the Great Lakes Watershed, 1837-Present

This visualization of the origins of invasive species in the Great Lakes watershed uses data from the NOAA Great Lakes Aquatic Nonindgineous Species Information System and was developed in Tableau.

The World Bank - Global Indicators, 2000-2012

This interactive web app allows the user to compare global development indicators for nations in the first decade of the 21st century. It was developed using R with data from The World Bank.
